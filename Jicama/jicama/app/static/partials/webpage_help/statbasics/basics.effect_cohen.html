Effect Size and Cohen’s d and f Values

The term "effect size" refers to the magnitude of a difference in outcomes that you are studying. An effect size is a numerical value whose units are the same as the outcome measure you are using to conduct your study.

You may be familiar with Cohen’s effect sizes (Cohen, 1988), especially Cohen’s d for t-tests and Cohen’s f for ANOVAs. These quantities are "relative" effect sizes: roughly speaking, Cohen’s d and f values correspond to the ratios of effect size to outcome standard deviation for T-test and ANOVA test, respectively. Their use in MySampleSize requires a bit of arithmetic that we can do for you below.

Explore Cohen's d and f values below!

Cohen's d

One-sample, two-sample, or paired T-tests
Calculate:

\text{EffectSize} = \left ( \text{standard deviation} \right) * \left ( \text{Cohen's } d \right)

Alternatively, you can input the value of 1 for the standard deviation and Cohen’s d for the effect size in a T-test design to obtain sample size and/or power.

Cohen's f

For designs that have more than two groups, Cohen’s f is a common relative effect size. Cohen’s f can be used in designs with one or more experimental factors, for studying individual factor effects or interaction effects. Each factor has a number of levels, and the number of experimental groups is the product of the numbers of levels for each factor.

For a main, secondary, or tertiary factor in a more-than-two-groups experiment
Calculate:

\begin{aligned} \text{EffectSize} &= \left ( \text{standard deviation} \right) * \left ( \text{Cohen's } f \right) * \sqrt{ \frac{\text{nLevels}}{ \text{nLevels}-1 } } \end{aligned}

Alternatively, you can input the value of 1 for the standard deviation and Cohen’s d for the effect size in a t-test design to obtain sample size and/or power.

You may recall that an ANOVA with two groups and a two-sample T-test are equivalent. However, Cohen’s d and f are actually different numbers for these two tests. Here are the relationships for an ANOVA with exactly two groups:

\begin{aligned} \text{Cohen's } f &= \frac{1}{2} * \left( \text{Cohen's } d \right) \\ \text{effectSize} &= \frac{1}{\sqrt{2}} * (\text{Cohen's } d) * (\text{standard deviation}) \end{aligned}

For an interaction between two factors

For experiments seeking to detect an interaction between two factors, Cohen’s f may also be used.

Calculate:

\begin{aligned} \text{EffectSize} &= \left ( \text{standard deviation} \right) * \left ( \text{Cohen's } f \right) \\ &* \sqrt{ \frac{ \text{nLevels}_1 * \text{nLevels}_2 }{ (\text{nLevels}_1-1) * (\text{nLevels}_2-1) } } \end{aligned}

For a three-way interaction

For experiments seeking to detect a three-way interaction between two factors, Cohen’s f is still the relative effect size commonly used.

Calculate:

\begin{aligned} \text{EffectSize} &= \left ( \text{standard deviation} \right) * \left ( \text{Cohen's } f \right) \\ &* \sqrt{ \frac{ \text{nLevels}_1 * \text{nLevels}_2 * \text{nLevels}_3 }{ (\text{nLevels}_1-1) * (\text{nLevels}_2-1) * (\text{nLevels}_3-1) } } \end{aligned}

Like means, standard deviations, and proportions, there are "true" or population effect sizes as well as "estimated" or sample effect sizes. In the planning stage, a researcher needs an idea of the smallest effect size that would be scientifically or clinically relevant if discovered. A strong design will have enough subjects to detect this minimally relevant effect size with sufficient power.
