Groups, Factors, Levels, Settings

A group in an experiment is a collection of subjects that are identical except only for subject-to-subject variation. Each subject in the group receives the exact same treatment setting (or control). Each subject in the group has the same non-treatment factor level. The number of subjects in each group is the quantity one seeks when determining sample size. The total number of subjects is then the number of subjects per group times the number of groups.


Determining the number of groups involves the number of levels of each factor. For example, for a study with the structure

Treatment 1 =
Drug A + Control
Treatment 2 =
Drug B (Setting 1) + Drug B (Setting 2) + Control
Nontreatment 1 =
Female (Level 1) + Male (Level 2)

TOTAL LEVELS for FACTOR 1 (Treatment 1) = 2 Levels

TOTAL LEVELS for FACTOR 2 (Treatment 2) = 3 Levels

TOTAL LEVELS for FACTOR 3 (Nontreatment 1) = 2 Levels

The total number of groups is 2 X 3 X 2 = 12. If this study has 8 subjects per group, then the total sample size is 8 X 12 = 96.
