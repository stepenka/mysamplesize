Latin Square

A Latin Square design consists of 2 blocking factors usually structured in rows and columns.

This design is very similar to a Randomized Block design but consists of a second blocking factor. Each blocking factor is usually a random effect and the treatment is one or more fixed effects. This is a very efficient design especially if there are many levels to each blocking factor.

  • The number of treatments must be the same as the number of replicates.

example

A researcher is interested in testing the effect of 4 different diets. One blocking factor will be the individual subjects to control for physical differences between subjects. The second blocking factor is diet. Each individual is given a different diet to follow for one month over the course of 4 months so that each individual is given each diet at some point.

Table of experimental design, filled in by diet type 1, 2, 3, or 4;

Month 1 Month 2 Month 3 Month 4
Subject 1 1 2 3 4
Subject 2 2 1 4 3
Subject 3 3 4 1 2
Subject 4 4 3 2 1

statistical tests

Statistical tests that are commonly appropriate for this design:

  • 3-Way ANOVA
  • Regression
