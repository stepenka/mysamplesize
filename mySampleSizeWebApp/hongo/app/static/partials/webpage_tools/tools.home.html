Quantifying your design involves balancing sample size, statistical power, and the minimal scientifically relevant effect you are seeking to detect.
These tools allow you to approach this problem in different ways.

Sample Size

Calculate Sample Size

Input your desired statistical power and relative effect size to determine the appropriate number of samples for your experiment.

Statistical Power

Calculate Statistical Power

If your sample size is constrained, you may need to find the statistical power that is achievable for that size and a relative effect size. This tool provides that calculation.

Power Plots

generate a graph of your calculations

If you would like to see graphs that show the dependence of sample size on statistical power and relative effect size, this tool will help.

Simulations

Visualizing experiments

Use this tool to simulate a random experiment and visualize what the data might look like.

Random Number Generator

Assign subjects based on design

Input your design to obtain a randomization table to ensure your experimental units are randomly assigned to groups.

Data Analysis

Visualize your data

Input up to three-way ANOVA data you provide to calculate p-value and view your dataset.

Summary Report

A downloadable summary of the tools used

After using our Tools, you can create a downloadable report summarizing the calculations and graphs.
